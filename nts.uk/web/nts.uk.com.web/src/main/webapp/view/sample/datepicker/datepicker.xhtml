API

Two mode: datepicker and yearmonth picker The second mode will not have 'startDate' and 'endDate' option.

Binding

Description
value
(optional)		 Type: Date | string
Default: today
The Date value for datepicker mode or string value for yearmonth picker mode
dateFormat Type: string
Optional Specify the desire format: 'yyyy/MM/dd' for simple date, 'yyyy/MM/dd DDD' or 'yyyy/MM/dd D' for date with Day name.
Or 'yyyy/MM' for yearmonth mode only
disabled Type: boolean
Optional Disable date picker
buttons Type: boolean
Optional: provide button for trigger datepicker
startDate Type: string
Optional: If presented, all date before this day will be disabled
endDate Type: string
Optional: If presented, all date after this day will be disabled
autoHide Type: boolean
Optional: default true, determine that the datepicker should be hide or not after a date selection click
