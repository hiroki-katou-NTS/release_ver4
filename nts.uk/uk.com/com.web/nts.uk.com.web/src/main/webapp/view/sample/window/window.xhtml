Overview

It's the current window you're working with. It has some useful API to share data between screen or close the current SubWindow.

Methods

Function

Description
setShared(key, data, isPersist) Return: void
key: string Set the key for stored share value
data: any Set data for stored share value
isPersist: boolean If true, data will be stored for this screen and all of dialog can use until screen is closed or jump to other screen. If false, data will be removed after that dialog is closed. In screen case, isPersist is true or false, it also same logic.
Store a share pair <key,value>
