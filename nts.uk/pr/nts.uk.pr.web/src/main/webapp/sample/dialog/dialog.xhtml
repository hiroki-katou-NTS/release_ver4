API

Options

Description
title Type: string
Default: ""
Title for dialog
message Type: string
Default: ""
Message content for dialog

Methods

Description
open() Show dialog
hide() Hide dialog

Events

Description
okButtonClicked() Triggered when OK button is clicked
cancelButtonClicked() Triggered when cancel button is clicked
