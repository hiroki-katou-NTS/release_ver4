TextEditor API

TextEditorOption

Description
textmode Type: string
Default: "text"
Members:
text: Single line text editor type text
password: Single line text editor type password
placeholder Type: string
Default: ""
Hint text for the control
width Type: string
Default: ""
Set width for the control
textalign Type: string
Default: "left"
Members:
left: Align text to left
right: Align text to right
center: Align text to center

Methods

Description
enable(boolean) Enable or disable control
readonly(boolean) Toggles the readonly state of the control

Events

Description
change(value) value: string
value is the control new value
Triggered when value of the control is changed
