Window

Overview

It's the current window you're working with. It has some useful API to share data between screen or close the current SubWindow.

Function

Description
modal(url, option?) Return: ScreenWindow
url: string Relative URL usually start with "/view"
option: object Dialog option extend from JqueryUI Dialog option
Open a modal sub-window, which you can't interact with parent window.
modeless(url, option?) Return: ScreenWindow
url: string Relative URL usually start with "/view"
option: object Dialog option extend from JqueryUI Dialog option
